# IIITA_ERP_Password_Tester

## Description

This Node.js application checks for all birth dates from 2003 to 2004 for a provided username (Application No). The script tests these dates to see if they match the usernameâ€™s password.
If the user hasn't changed ERP password (which by default is date of birth), this will output the correct password

## Prerequisites

- Node.js (v12 or higher)
- npm (Node Package Manager)

## Installation
 ```sh
npm i && node index.js <Application_number>
 ```
## Example
```sh
npm i && node index.js IIT2022000
 ```
